# spring-boot-react-enterprise
Enterprise-level EMS platform featuring a React frontend and Spring Boot REST API. Built with a modular monorepo structure, JPA/Hibernate, and scalable state management.

# Enterprise Management System (EMS)

A full-stack enterprise application utilizing a decoupled architecture.

## 🏗️ Tech Stack
- **Frontend:** React 18+, Tailwind CSS, Redux Toolkit
- **Backend:** Spring Boot 3.x, Spring Security, Hibernate
- **Database:** PostgreSQL / MySQL

## 📁 Project Structure
- `/client` - React frontend (Vite/CRA)
- `/server` - Spring Boot backend (Maven/Gradle)
